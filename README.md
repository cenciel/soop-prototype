## 🎨 Prototypes

### SOOP UI
라이브 스트리밍 플랫폼 SOOP의 모바일 UI 프로토타입

**기술 스택**
- React 18.3.1
- Vite 6.3.5
- Tailwind CSS 4.1.12
- Radix UI

**주요 기능**
- 라이브 스트림 카드
- 카테고리 브라우징
- 추천 스트리머 (Follow 기능)
- 라이브 플레이어 & 채팅
- 어두운 테마

## 🛠️ 설치 및 실행

```bash
# 의존성 설치
cd prototypes/soop-ui
npm install

# 개발 서버 실행
npm run dev

# 프로덕션 빌드
npm run build

# Firebase 배포
firebase deploy
```
